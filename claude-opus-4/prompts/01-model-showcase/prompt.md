# Промпт 01 — model-showcase (claude-opus-4)

Исторический журнал: дословный промпт, на который отвечала модель
`claude-opus-4-8` в ходе трека. Сам тест-шаблон и критерии живут в
общем каталоге [`prompts/01-model-showcase/`](../../../prompts/01-model-showcase/).

## Дословный промпт

> сделай плз папку промпты в руте там как раз будет общие тесты.
> котоыре можно переписпользовать. первый это будет сайт про саму
> модель 0 рассказать что за модель и чем хороша. на выходе один
> html файл. использовать диграммы, схемы, интерактивные виджеты
> и тексты которые нельзя назвать ии слопом.

## Контекст запуска

- **Модель**: Claude Opus 4.8 (`claude-opus-4-8`), 1M-context вариант
  (`claude-opus-4-8[1m]`), запущена как Claude Code в этой сессии.
- **Дата**: 2026-06-14.
- **Что делает модель**: презентует сама себя — одностраничный
  self-contained HTML про Opus 4.8.
- **Опечатки** в промпте (`котоыре`, `переписпользовать`, `диграммы`,
  `слопом`) сохранены дословно — тест про результат, а не про
  редактуру промпта.
