# Тест 02 — 3D shooter game

Минимально приемлемый 3D-шутер от первого лица. На входе —
дословный user-промпт из трёх слов, на выходе — запускаемый
проект с 3D-сценой, FPS-управлением, стрельбой и хотя бы одним
противником с ИИ.

## Вход

Дословный промпт в [`prompt.md`](./prompt.md). Контекст — в том же
файле.

## Что НЕ указано в промпте

Специально. Никакого стека, никакого движка, никакого формата
сборки. Модель **сама** выбирает:
- 3D-движок / библиотеку
- Способ сборки и запуска
- Стилистику графики
- Жанр и набор механик внутри шутера

Это часть теста — свобода выбора. По готовому артефакту видно,
насколько модель умеет принимать разумные дефолты.

## Ожидаемый артефакт

Запускаемый проект (структура и стек — на усмотрение модели):
- `index.html` или эквивалент точки входа
- исходники в логически выделенных файлах/папках
- без внешних ассетов
- headless smoke, проходящий критерии из `criteria.md`

## Критерии оценки

В [`criteria.md`](./criteria.md). Единый чеклист для всех моделей.
Критерии сформулированы в терминах **результата**, а не
инструментов.

## Примеры

Пусто. Ни одна модель пока не прогоняла этот тест «вслепую» (без
подсказок о стеке). См. [`examples/README.md`](./examples/README.md).

## Как запустить тест на новой модели

1. Открой новую сессию с целевой моделью.
2. Скопируй дословный текст из `prompt.md` и отправь модели.
3. Модель, скорее всего, начнёт спрашивать детали. **Не отвечай** —
   дай ей самой принять разумные дефолты. Если она зависает без
   ответа, только тогда задай уточняющий вопрос.
4. Получи артефакт (запускаемый проект).
5. Прогони `criteria.md` по результату. Зафиксируй оценку.
6. Положи артефакт в `examples/<model-slug>/`.
7. Сделай скриншоты через `smoke.mjs`, положи в
   `examples/<model-slug>/screenshots/`.
8. Напиши `examples/<model-slug>/README.md`: модель, оценка,
   smoke-результат, ссылка на issue.
9. Создай child issue от трека модели по правилам `manager`.

## Чего этот тест НЕ измеряет

- Какой именно стек выбрала модель. Это часть теста.
- Качество графики.
- Скорость работы.
- Реалистичность стрельбы.
- Мультиплеер.
